# 🌌 PomoSpace

PomoSpace no es solo otro temporizador Pomodoro; es un espacio de trabajo diseñado con una estética "Glassmorphism" de alta gama, brillos de neón vibrantes y animaciones fluidas para que tus sesiones de enfoque sean tan agradables a la vista como efectivas.

---

## ✨ Características

### ⏳ Temporizador Pomodoro Avanzado
- **Visuales Estéticos**: Gradientes radiales profundos y superficies de cristal (glassmorphism) real.
- **Pulso de Neón**: El anillo del temporizador tiene un brillo de neón tipo "respiración" que se mantiene activo durante la sesión.
- **Notificaciones de Onda**: Un efecto visual de onda ("ripple") se activa al completar cada ciclo.
- **Sonidos Personalizados**: Elige tus propios sonidos de notificación desde tu dispositivo.

### 📋 Gestión de Actividades Inteligente
- **Tareas Anidadas**: Divide grandes objetivos en sub-actividades para un control granular.
- **Seguimiento de Progreso**: Barras de progreso en tiempo real para cada tarea principal basadas en la finalización de sub-tareas.
- **Tarjetas Expandibles**: Interfaz organizada y limpia con detalles de tareas expandibles.

### 🌐 Soporte Multilingüe
- Totalmente traducido al **Inglés** y **Español**.
- Actualizaciones dinámicas de la interfaz sin reiniciar la aplicación.

---

## 🛠️ Stack Tecnológico

- **Framework**: [Flutter](https://flutter.dev/) 🚀
- **Gestión de Estado**: StatefulWidget con persistencia mediante SharedPreferences.
- **Audio**: `audioplayers` para una experiencia inmersiva.
- **Diseño**: Google Fonts (Outfit), CustomPainters para efectos de onda y uso intensivo de BackdropFilters.

## 🚀 Comenzando

### Instalación

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/tuusuario/pomospace.git
   ```
2. **Instalar dependencias**:
   ```bash
   flutter pub get
   ```
3. **Ejecutar la app**:
   ```bash
   flutter run
   ```

### Generar APK para Android

Para generar un APK de lanzamiento:
```bash
flutter build apk --release
```
